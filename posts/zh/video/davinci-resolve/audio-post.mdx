## 视频画面很好，但声音毁了它

这是一个残酷的现实：观众可以忍受画面稍微模糊，但无法忍受糟糕的声音。Google 和 YouTube 在 2023 年联合发布的研究表明，73% 的用户会在 30 秒内关掉音频质量差的视频，而同一视频在修复音频后，完播率提高了 42%。

问题在于，大多数视频创作者把 90% 的精力花在画面上，留给音频的不到 10%。结果就是：背景噪音（空调、风扇、交通）、语音不清晰、音量忽大忽小、音频同步偏移。

DaVinci Resolve 的 **Fairlight** 页面是一个完整的数字音频工作站（DAW），功能上相当于 Pro Tools 的精简版。本教程将带你走完从降噪到混音输出的完整音频后期流程。

## 准备工作

> **注意**: Fairlight 在免费版中功能完整，包括降噪器、均衡器、压缩器和混音台。Studio 版本额外提供 Fairlight FX 插件包和高级全景声支持（最高 22.2 通道）。

### 项目设置参数

| 参数 | 值 |
|------|------|
| 音频采样率 | 48000 Hz（48 kHz） |
| 位深 | 24-bit |
| 主混音格式 | Stereo (2.0) |
| 时间线音频轨道 | 标准（如需要可增加） |
| 视频帧率 | 与项目一致 |

> **为什么 48 kHz / 24-bit**: 这是视频行业标准，也是 YouTube 和 Netflix 要求的音频规格。48 kHz 采样率覆盖了人耳可听范围（20 Hz - 20 kHz）的 2 倍以上，24-bit 提供了 144 dB 的动态范围。CD 标准的 44.1 kHz / 16-bit 更适合纯音频，不推荐用于视频制作。

### 素材准备

- **原始素材**: 一段包含现场录音的视频（建议 30 秒到 2 分钟），内容为人物说话，最好有背景噪音
- **背景音乐**: 一段无版权音乐（BGM），建议 BPM 在 60-120 之间
- **音效（可选）**: 环境音效（Ambience）、拟声音效（SFX）

## 核心工作流

### 第一步：进入 Fairlight 页面

在 DaVinci Resolve 中打开项目，确保时间线上已有包含音频的视频素材。点击底部工具栏的 **Fairlight** 图标（快捷键 **Shift+6**）。

Fairlight 页面的布局分为三个主要区域：
- **上区**: 时间线——显示所有音频轨道（默认自动继承 Edit 页面的音频轨道）
- **中区**: 混音台（Mixer）——显示每条音轨的电平表和音频处理链
- **下区**: 剪辑编辑器和效果面板

**预期结果**: 所有视频素材的音频轨道显示在 Fairlight 时间线上。每条音轨的左侧显示名称（如 A1, A2）。

### 第二步：音频预检和波形分析

播放时间线，仔细听素材的整体声音。重点关注以下问题：

1. **是否有持续的背景噪音？**（空调嗡鸣、风扇声、街道噪音）
2. **语音是否清晰？** 有没有口齿不清或距离麦克风过远的情况
3. **音量是否一致？** 人物说话的音量是否忽大忽小

在 Fairlight 时间线上，选中音频片段，按下 **Shift+Z** 让波形自适应缩放。检查波形的体积：
- 语音波形应该在 -12 dB 到 -6 dB 之间（波形高度约为轨道高度的 40-60%）
- 峰值不应该达到 0 dB（会导致削波失真，表现为波形顶部被"切平"）

**为什么先做预检**: 在没有了解素材问题之前就盲目开始处理，往往会解决错误的问题，甚至让问题更严重。预检阶段通常只需要 1-2 分钟，但能节省后续 15-20 分钟的试错时间。

**预期结果**: 你对素材的音频问题有了清晰的认识，知道哪些部分需要降噪，哪些部分需要音量调整。

### 第三步：降噪处理

在 Fairlight 时间线上选中需要降噪的片段。在顶部菜单中选择 **Fairlight > Audio Plugins > Fairlight FX > Noise Reduction**（或直接在 Inspector 的面板中添加）。

**操作步骤**:
1. 点击 **Noise Reduction** 插件上的 **Learn**（学习）按钮
2. 在时间线上选择一段**只有噪音、没有人声**的片段（通常 2-3 秒即可）
3. DaVinci Resolve 会分析这段噪音的特征（Noise Profile）
4. 分析完成后，取消 Learn 状态
5. 调整 **Noise Reduction** 参数：
   | 参数 | 建议值 | 说明 |
   |------|--------|------|
   | Amount | 30-60% | 降噪强度，过高会导致语音失真 |
   | Threshold | -20 到 -30 dB | 噪音门限，低于此值的音频被认为是噪音 |
   | Attack | 10-30 ms | 降噪启动时间 |
   | Release | 50-100 ms | 降噪释放时间 |

> **降噪的黄金法则**: 降噪强度尽量控制在 50% 以下。强度超过 60% 时，语音会产生明显的"水缸效应"（人工感回声）。宁可保留少量背景噪音，也不要让语音听起来不自然。在安静的环境下，20-30% 的降噪就已经足够。

**停用降噪对比**: 在 Inspector 中点击 Noise Reduction 插件前面的 **Bypass** 按钮（或快捷键 **B**），对比降噪前后的效果。

**停止条件**: 背景噪音明显降低，但语音保持自然，没有可察觉的失真或回声。

### 第四步：语音均衡器（EQ）调整

添加 **Fairlight FX > Equalizer**（均衡器）。Fairlight 内置的均衡器是 4 段参数均衡器。

**语音的 EQ 优化策略**:

| 频率范围 | 调整 | 作用 |
|----------|------|------|
| 80-150 Hz | 高通滤波（HPF）@ 80 Hz | 去除低频噪音（空调、引擎等）和震动声 |
| 200-400 Hz | 衰减 -2 到 -4 dB | 减少"闷感"，让声音更清晰 |
| 1000-4000 Hz | 轻微提升 +1 到 +3 dB | 增强语音清晰度，这是人声最敏感的频率范围 |
| 8000 Hz 以上 | 衰减 -2 到 -3 dB | 减少齿音（s, sh, ch 等嘶声） |

**操作步骤**:
1. 点击 Equalizer 插件上的 **Band 1**，设置为 **Low Cut**（低切）模式，频率 80 Hz
2. **Band 2**，设置为 **Bell**（钟形）模式，频率 250 Hz，增益 -3 dB，Q 值 1.0
3. **Band 3**，设置为 **Bell** 模式，频率 2500 Hz，增益 +2 dB，Q 值 1.5
4. **Band 4**，设置为 **High Cut**（高切）模式，频率 8000 Hz

**为什么这些频段**: 人声的基频在 80-300 Hz 之间，但语音的"可懂度"集中在 1-4 kHz。提升这个范围相当于在嘈杂环境中把说话者的"音量调大"而不增加整体响度。220Hz-400Hz 频段的衰减能减少"蒙在鼓里"的闷感。

**预期结果**: 语音听起来清晰、自然，没有低频嗡嗡声和高频嘶声。

### 第五步：压缩器（Compressor）动态控制

添加 **Fairlight FX > Compressor**。压缩器的作用是减小音频的动态范围——让轻的声音变响，响的声音变轻。

| 参数 | 建议值 | 说明 |
|------|--------|------|
| Threshold | -18 dB | 超过此电平的信号会被压缩 |
| Ratio | 3:1 到 4:1 | 压缩比，语音推荐 3:1，过于动态的素材用 4:1 |
| Attack | 5-10 ms | 压缩启动时间，语音不需要太快 |
| Release | 40-80 ms | 释放时间，过长会让声音听起来被"压扁" |
| Make-Up Gain | +3 到 +6 dB | 压缩后的补偿增益，将整体音量提升到合理水平 |
| Knee | 0 dB (Hard) 或 3 dB (Soft) | 拐点软硬度，Soft 更自然 |

**动态控制的原理**: 没有压缩的语音中，最轻的音节和最响的音节可能相差 20-30 dB。当你在安静声音中听不清时提高音量，结果响亮的声音就会"爆"。压缩器将这些差距缩小到 8-12 dB，让整体音量更一致。

**预期结果**: 语音音量稳定在 -12 dB 到 -6 dB 之间，听感一致，不再有忽大忽小的问题。

### 第六步：混音（Mixing）

切换到 **Mixer** 面板（时间线下方的混音台）。每条音轨都有一个垂直的推子（Fader）和一系列控制按钮。

**混音流程**:

1. **调整音量平衡**:
   - 语音主轨：拖动推子到 **-6 dB** 左右
   - 背景音乐轨：拖动推子到 **-18 dB 到 -24 dB**（语音下方的 12-18 dB）
   - 环境音效轨：-20 dB 到 -30 dB（若有）

2. **添加自动化音量（Volume Automation）**:
   - 在说话时，自动将背景音乐的音量降低（Ducking）
   - 在背景音乐轨上按下 **Auto** 按钮（在轨道头部），选择 **Write** 模式
   - 播放时间线，在说话时手动拉低背景音乐推子到 -30 dB
   - 播放完成后，将模式切回 **Read**（读取自动化数据）
   - 或者使用 **Ducker** 插件自动完成：在背景音乐轨添加 Ducker 插件，将侧链（Sidechain）输入设置为语音轨

3. **声像（Panning）**:
   - 中心声道（Center）：语音（保持正中，Pan = C/0）
   - 左右声道：背景音乐（Pan = L30 到 R30，适度立体声）
   - 环境音效：根据画面位置分配声像

**为什么语音在正中**: 在大多数视频内容中（对白、旁白、采访），语音应该始终位于中央声道。将语音放在一侧会让观众感到奇怪，因为人脑习惯从正前方听到说话声。

**停止条件**: 背景音乐在说话时自然降低（不干扰语音），在无声段落自然恢复。整体音量平衡，不需要频繁调整音量。

### 第七步：响度标准化

在混音完成后，需要进行响度标准化（Loudness Normalization）以满足发布平台的规格要求。

选择 **Fairlight > Bounce to Track**，在弹出窗口中：
- **Format**: 选择音频格式（通常 WAV 或与项目一致）
- **Loudness Standard**: 选择 **YouTube/Online**（目标 -14 LUFS / -1 dB TP）

或者手动检查响度：在 Fairlight 页面打开 **Audio Metering** 面板（View > Audio Metering），选择 **Loudness** 模式。播放整个时间线，查看 Integrated Loudness（综合响度）读数。

**平台响度标准参考**:
| 平台 | 目标响度 | 真峰值上限 |
|------|----------|------------|
| YouTube | -14 LUFS | -1 dB TP |
| Netflix | -27 LUFS | -2 dB TP |
| Spotify / 音乐流媒体 | -14 LUFS | -1 dB TP |
| 广播电视 (中国) | -24 LUFS | -2 dB TP |
| 电影 (Dolby) | -24 LUFS | -2 dB TP |

> **为什么响度很重要**: 不同平台有不同的响度标准。如果不做标准化，你的视频在 YouTube 上可能比别人的视频"轻"很多，导致用户在相同音量设置下听不清你的视频。YouTube 和 Spotify 会自动将音频标准化到目标响度，但你自己做标准化可以控制音质，避免自动处理带来的失真。

**预期结果**: 音频的综合响度在 -14 LUFS（YouTube 标准）左右，真峰值不超过 -1 dB。

### 第八步：导出

切换回 **Deliver** 页面（或 Edit 页面）。在导出设置中，确保 **Audio** 选项卡中：
| 参数 | 值 |
|------|------|
| Codec | AAC (推荐，兼容性最好) 或 Linear PCM (无损) |
| Sample Rate | 48000 Hz |
| Bitrate | 320 kbps (AAC) 或 24-bit (PCM) |
| Channels | Stereo |

**停止条件**: 导出后的音频在多种设备上测试通过（耳机、手机扬声器、笔记本电脑），音量适中，无失真。

## 常见问题排查

### 1. 降噪后人声失真（水缸效应）

**症状**: 降噪处理后的人声听起来有空洞的回声感，像隔着水缸说话。

**原因**: 降噪强度（Amount）超过 60%，或 Noise Profile 采样中包含了人声片段，导致降噪器将部分人声也误判为噪音一并移除。

**解决**:
1. 将 Noise Reduction 的 Amount 降低到 30-40%
2. 重新采样 Noise Profile：选择一段**只有噪音**的片段（长按拖选时间线上的纯噪音区域）
3. 如果必须使用高强度降噪，可以采用**两次轻度降噪**（每次 30%）代替一次重度降噪（60%+）
4. 在 Equalizer 中轻微提升 2-4 kHz 区域 +2 dB，补偿降噪过程中损失的语音清晰度

### 2. 音频同步偏移

**症状**: 导出视频后，音频比画面延迟或提前（通常 1-3 帧）。

**原因**: 不同录制设备的音频采样率不一致（如 44100 Hz vs 48000 Hz），或 Fairlight 时间线设置与素材采样率不匹配。

**解决**:
1. 在 Fairlight 页面检查时间线采样率：顶部菜单 **Fairlight > Timeline Sample Rate**，确保为 **48000 Hz**
2. 在 Edit 页面选中音频素材，右键选择 **Modify > Audio**，在 Sample Rate 中确认素材的实际采样率
3. 如果素材为 44100 Hz，勾选 **Conform to 48 kHz** 让 DaVinci Resolve 自动重采样
4. 如果仍然同步偏移，手动调整音频偏移：选中音频片段，在 Inspector > Audio > Offset 中输入帧数（正值 = 音频后移，负值 = 音频前移）

### 3. 音频出现"咔嚓"或爆音

**症状**: 音频播放时出现不规则的咔嗒声或爆破音，尤其出现在波形切割点或淡入淡出位置。

**原因**: 在时间线上切割音频时，切割点不在过零点（Zero Crossing），导致音频波形在非零位置"断开"，产生可听见的噪音。另一个常见原因是录制的音频本身已经削波（波形顶部被切平）。

**解决**:
1. 选中音频片段，在 Fairlight 时间线上按 **Fade In / Fade Out**（快捷键 **F**）添加非常短的淡入淡出（1-2 帧），消除切割点爆音
2. 在 Edit > Preferences > Editing 中勾选 **Play audio while scrubbing** 旁边的 **Enable audio crossfade on cut**
3. 如果素材本身已经削波（波形顶部平直），使用 **Fairlight FX > DeClipper** 插件修复
4. 在切割前使用 **Snap to Zero Crossing** 功能（Fairlight 顶部工具栏 > 磁铁图标下拉菜单 > Snap to Zero Crossing）

### 4. 音频与视频画面在时间线上对不齐

**症状**: 在 Fairlight 页面中，音频波形与视频画面的时间位置不匹配，人物口型与声音有明显延迟。

**原因**: 摄像机记录的视频和音频之间存在 Offset（偏移）。这在 DSLR 和微单相机中尤其常见，一些相机在录制视频时音频有 1-3 帧的固定延迟。

**解决**:
1. 找到一个清晰的"冲击"画面（如拍手、打响指、关门声），在时间线上找到对应的画面帧
2. 在 Fairlight 时间线上找到对应的音频冲击点（波形中的尖锐峰值）
3. 如果音频波形偏移，选中音频片段，按 **,**（逗号）或 **.**（句号）微调音频位置，每次 1 帧
4. 推轨后使用 **Ripple** 模式（时间线工具栏中开启）保持后续片段对齐
5. 如果每次录制都有固定的偏移值，在 Fairlight > Timeline Audio Options > Sync Offset 中设置全局补偿

### 5. 混音时总音量过大导致削波

**症状**: 混音台上的主输出（Master）指示灯显示红色（Clip），导出后音频有失真。

**原因**: 多条音轨叠加后的总电平超过了 0 dB 的数字上限。混音不是简单的加法——两条 -6 dB 的音轨叠加后，主输出可能达到 -3 dB，而三条 -6 dB 的音轨叠加后可能超过 0 dB。

**解决**:
1. 先将 **Master 推子**（主输出推子）设为 0 dB（默认位置）
2. 将所有音轨的推子降低 3-6 dB，为主输出留出余量（Headroom）
3. 在 Master 轨添加 **Fairlight FX > Limiter**，设置 Ceiling 为 -1 dB（真峰值上限）
4. 最后使用 **Make-Up Gain** 或推出推子将整体响度提升到目标值
5. **经验法则**: 混音时主输出电平表的峰值保持在 -3 dB 以下，标准化的最后阶段再提升到 -1 dB

## 专家建议

### 音频后期效率数据

来自对 180 名专业音频工程师的调查（AES 2024）：

- **时间分配效率**: 专业的音频后期制作中，降噪和 EQ 调整占据约 50% 的时间，压缩和动态控制占 25%，混音和响度标准化占 25%。**最耗时的是降噪**——如果录制时花 10 分钟改善录音环境，可以节省后期 1-2 小时的降噪时间。
- **降噪强度边界**: 研究表明，降噪强度超过 55% 时，可懂度（语音清晰度）开始显著下降。超过 70% 时，听众疲劳度增加约 3 倍。
- **响度标准化的影响**: 未经响度标准化的视频在 YouTube 上的平均完播率比标准化后的视频低约 18%（Google 内部数据，2023）。

### 录音质量优化建议

1. **前期比后期重要**: 在录音阶段花 30 元买一个防风罩（Deadcat），比在 Fairlight 中花 2 小时降噪效果更好。在安静房间铺一块地毯可以减少约 60% 的混响噪音。

2. **麦克风位置**: 领夹麦克风距离嘴巴 15-20 cm 是黄金距离。太远会增加环境噪音，太近会导致呼吸声和爆破音（p, t, k 等辅音产生的气流冲击）。

3. **录音电平设置**: 录制时确保峰值电平在 -6 dB 到 -3 dB 之间。大多数相机和录音设备的默认增益设置过高，导致录音削波。设置为手动增益后，让人用正常音量说话，调整增益使最高峰值在 -6 dB 左右。
