> **版本**: Blender 4.0+（Windows / macOS / Linux）
> **难度**: ★★☆☆☆（有基本电脑软件操作经验）
> **用时**: 25-30 分钟

## 打开 Blender 的那一刻，你可能想关掉它

第一次打开 Blender 时，你看到的界面是这样的：深灰色背景、一个立方体、三个轴箭头、以及几十个不知道干什么用的按钮。三秒后你想关掉它——"这也太复杂了。"

**你不是一个人**。Blender 以其陡峭的学习曲线著称，但它也是目前**功能最全且完全免费**的 3D 创作套件。根据 Blender Foundation 2023 年的数据，Blender 在全球有超过 **1400 万用户**，在独立游戏开发和独立动画制作领域已经成为主流选择。本教程将带你走过最关键的"前 30 分钟"——从打开软件到完成你的第一个完整建模作品。

## 准备工作

- **软件**: Blender 4.0 或更高版本，从 [blender.org](https://www.blender.org/) 免费下载
- **账号**: 不需要任何账号，Blender 完全离线可用
- **文件**: 无需现有文件——我们将从默认场景（含立方体）开始
- **前置知识**: 不需要任何 3D 经验。需要基本的 3 键鼠标（左键、右键、滚轮）

## 核心工作流

### 第一步：界面导航——"你只需要三个手势"

Blender 的 3D 视图导航只需要鼠标三个操作：

1. **旋转视图**: 按住鼠标中键（滚轮）拖动
2. **平移视图**: 按住 `Shift` + 鼠标中键拖动
3. **缩放视图**: 滚动鼠标滚轮

**快速定位技巧**: 如果滚着滚着找不到物体了，按小键盘 `.` 键（句号键）——视图会立即聚焦到选中的物体上。没有小键盘？在 **编辑 → 偏好设置 → 输入**中勾选"模拟数字小键盘"

**为什么这样做？** 这三种操作覆盖了你在 3D 空间中需要的所有视角控制。不要用工具栏里的"缩放"按钮——效率远低于滚轮。熟练后，你会本能地用鼠标中键旋转视角来检查模型的每个角度。

**停止条件**: 你可以在 3 秒内将视图从俯视切换到侧视再切换到透视——流畅无卡顿。

### 第二步：选择和变换——移动/旋转/缩放的基础

现在开始操作那个默认立方体：

1. **选中立方体**: 用鼠标**右键**点击立方体（Blender 默认右键选择，与大多数软件不同）。选中后物体边缘显示橙色高亮
2. **移动（Grab）**: 按 `G` 键进入移动模式——移动鼠标拖动立方体，点击左键确认位置，点击右键取消
3. **约束移动**: 按 `G` 后按 `X` / `Y` / `Z` ——物体只在对应轴向上移动。这是最常用的组合键
4. **旋转**: 按 `R` 键进入旋转模式，按 `R` 再按 `X` / `Y` / `Z` 约束旋转轴
5. **缩放**: 按 `S` 键，同样可以加 `X` / `Y` / `Z` 约束

**为什么这样做？** `G`（Grab）/ `R`（Rotate）/ `S`（Scale）是 3D 建模中最频繁使用的操作。约束轴（按 `G` 再按 `X`）确保物体沿一个方向移动，不会歪到一边。这与 2D 设计中的 `Shift` 约束角度是同一个逻辑。

**停止条件**: 你已成功将立方体在 Z 轴上移动 5 个单位、绕 Y 轴旋转 45°、沿 X 轴缩放至 2 倍。如果数字面板可见，位置坐标显示为 (0, 0, 5)，旋转 (0°, 45°, 0°)，缩放 (2, 1, 1)。

### 第三步：编辑模式——从"立方体"到"物体"的关键

默认立方体只是一个长方体。真正的建模从编辑模式开始：

1. 选中立方体，按 `Tab` 键切换到 **编辑模式**
2. 你现在看到的是点（顶点）、线（边）、面三个层级
3. 按 `1` / `2` / `3` 分别切换顶点选择、边选择和面选择模式
4. **挤出面**: 选中一个面，按 `E` 键——向外拖动拉出新的几何体。这就是"挤出建模"的核心操作
5. **内插面**: 选中一个面，按 `I` 键——在面内部创建缩小的新面，这是挖洞的前置操作

**为什么这样做？** 挤出（`E`）是建模中最重要的单个操作。从立方体挤出一个椅子腿、从球体挤出动物嘴巴、从平面挤出地形——几乎所有 3D 模型都离不开挤出。

**停止条件**: 一个立方体经过 3 次挤出操作后变成了一个 L 形或 T 形结构。

### 第四步：使用修改器——非破坏性的力量

修改器是你最强大的建模工具。它们像 Photoshop 的调整图层——不破坏原始几何体，随时可以修改或关闭：

1. 在编辑模式下选中模型，按 `Tab` 回到物体模式
2. 点击右侧属性面板的 **扳手图标**（修改器属性）
3. 点击 **添加修改器**，选择以下最常用的三个：

**细分表面（Subdivision Surface）**:
- 设置 Levels Viewport: 2
- 立方体瞬间变成圆滑的球体！但边缘太圆了
- 在编辑模式下选中一条边 → `Ctrl+E` → 标记为锐边（Mark Sharp）——边缘变硬

**倒角（Bevel）**:
- 设置 Width: 0.05m，Segments: 3
- 让尖锐边缘变成平滑倒角——这是"工业设计感"的秘密

**镜像（Mirror）**:
- 设置 Axis: X
- 删除立方体一半，修改器自动对称复制——做角色或对称物体时必用

**为什么这样做？** 细分表面用数学算法增加几何细节——Level 1 将面数翻 4 倍，Level 2 翻 16 倍。但你永远可以"撤销"这个平滑效果（删除修改器即可），因为原始几何没有被改变。

**停止条件**: 立方体应用了细分表面修改器后变成了一个光滑的球体，且你能恢复它的原始形态。

### 第五步：创建你自己的模型——一张桌子

将所有技能整合起来做一个简单桌子：

1. 删除默认立方体：按 `A` 全选，按 `X` → 删除（或 `Delete` 键）
2. 按 `Shift+A` → **网格 → 立方体**，创建一个新立方体
3. 按 `S` 然后按 `Z` 缩放至 0.1——这是桌面？不，这是桌腿
4. 在 Z 轴上缩放至 1.5（拉高），这将是桌腿
5. **制作桌腿**: 长方体桌腿（桌腿 0.1×0.1×1.5，离地面 0）
6. 按 `Shift+A` 再添加一个立方体，缩放至 2×0.05×1，这将是桌面，放在 Y 轴 -0.5 位置
7. 选中桌面，添加 **细分表面** 修改器（Level 1）——桌面边缘变得圆润
8. 选中四条桌腿和桌面，按 `Ctrl+J` 合并为一个物体

**为什么这样做？** 桌子是一个多物体组合建模的练习。实际建模流程就是"分而治之"——分别创建每个部件，调整到正确位置，最后合并。

**停止条件**: 一张由四个长方体桌腿和一个长方体桌面构成的桌子，桌面有轻微倒角边缘。

### 第六步：基本材质——给模型上色

没有材质的模型是灰色的。用简单材质让它"活"起来：

1. 选中桌子，点击右侧属性面板的 **红色球体图标**（材质属性）
2. 点击 **新建** 创建材质
3. 在 Shader Editor 面板中（也可以在下拉菜单中选择 Shading 工作区）：
   - **基础色（Base Color）**: 点击颜色框，选择木质棕色 `#8B6914`
   - **粗糙度（Roughness）**: 设为 0.4（木头是半粗糙表面）
   - **金属度（Metallic）**: 设为 0（木材不是金属）
4. 为桌面单独创建材质：在编辑模式下选中桌面顶部，在材质面板点击 **+** 新建材质，点击 **指定**（Assign）

**为什么这样说对？** Roughness 0.4 模拟了木材表面的半哑光质感。Roughness 0 是完全镜面（玻璃），1 是完全粗糙（砂纸）。大多数自然材质在 0.2-0.8 之间。记住这个范围可以快速调出真实感。

**停止条件**: 桌腿显示为深木色，桌面显示为浅木色（或不同颜色）。在渲染预览模式（按 `Z` → 选择"渲染"）下查看。

## 常见问题

### 物体在编辑模式下变成了透明线框

**原因**: 你切换到了线框显示模式（Wireframe），或者意外按到了 `Z` 键后选择了不同的显示选项。
**解决**: 按 `Z` 键打开饼图菜单，选择 **实体（Solid）**——左上角四个选项中的第二个。或者直接按 `Shift+Z` 切换回实体显示。

### 挤出操作后新面与旧面分离

**原因**: 挤出前没有选中整个面——只选中了面的部分顶点，导致挤出产生了悬挂的孤立面。
**解决**: 按 `3` 切换到面选择模式，确保点击面的中心点（高亮整个面而不是顶点）后再按 `E` 挤出。如果不慎挤错了，按 `Ctrl+Z` 撤销后重新操作。

### 修改细分表面后电脑变得很卡

**原因**: 细分表面的视图细分值（Viewport Levels）设置过高。Level 3 以上会生成数百万个面，你的显卡在实时计算。
**解决**: 在修改器面板中将 Levels Viewport 临时降为 1，需要渲染时再提高。渲染时使用 Levels Render（渲染细分值），这个值不影响视口性能。

### 右键选择的物体没有高亮

**原因**: Blender 默认的选择颜色是橙色，但在某些自定义主题下可能不明显。或者物体被其他物体遮挡了。
**解决**: 在 **编辑 → 偏好设置 → 主题 → 3D 视图** 中调整"选择高亮颜色"。尝试改为亮青色或亮绿色以便辨认识别。如果物体被遮挡，按 `Z` → 选择"线框"模式查看。

## 专家技巧

- **快速摄像机视角**: 按小键盘 `0` 切换到摄像机视角。按 `Ctrl+Alt+0` 将当前视图设为摄像机位置——这比手动调整摄像机快 10 倍。
- **环切与循环选择**: 按 `Ctrl+R` 在模型上加一圈循环边，滚动滚轮可以增加切刀数量。按 `Alt`+点击一条边可以选中整圈循环边——这在选择复杂模型的特定区域时极为高效。
- **平滑着色**: 建模完觉得模型面感太重？在物体模式下，右键 → **平滑着色（Shade Smooth）**。如果觉得太模糊，在修改器属性中添加 **边缘分割（Edge Split）** 修改器，保持锐边清晰。
- **吸附功能**: 移动物体时按住 `Ctrl` 启用吸附（Snap），默认吸附到网格。这在需要精确对齐时非常有用。也可以在工具栏中切换吸附类型（顶点、边、面）。
- **Blender 资源库**: 按 `Shift+A` 后 → "网格"下方还有"场景 kit"和"背景 kit"——这些是 Blender 4.0 内置的免费资源库，包含家具、植物等预置模型，可直接使用。

## 总结

你已在 Blender 中走完了完整的建模流程：界面导航 → 物体变换 → 编辑模式挤出 → 修改器应用 → 组合建模 → 材质上色。核心收获是理解"从基本几何体出发，通过挤出和修改器构建复杂形状"的思路。

接下来可以探索纹理贴图（UV Unwrapping）、骨骼绑定（Armature）或学习在 Blender 中做简单的动画。Blender 官方文档和 YouTube 的 Blender Guru 频道是很好的进阶学习资源。
