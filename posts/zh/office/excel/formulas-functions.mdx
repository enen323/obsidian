> **难度：** ★★★★☆（4/5）  
> **预计用时：** 45-60 分钟

你的工作表中有一万行销售记录。经理问："A 产品第三季度的总收入是多少，扣除退货？" 你盯着数据，知道答案就藏在那些行里，但如果手动筛选，几个小时就没了。复制粘贴、排序、手动计算，耗尽了一个下午——而且数字还对不上。

大多数初学者靠死记硬背几个函数来应付 Excel。问题在于：如果不理解 Excel 公式引擎的工作原理，你写出来的公式就会很脆弱——数据一变就出错。一个美元符号放错位置或括号不匹配，就可能导致整份报告出错。

本教程不只是罗列函数。它教你 **公式构建的思维模式**：如何用查找函数读取数据、用条件聚合进行汇总、用文本和日期函数处理非结构化数据、用 IF 嵌套实现业务逻辑判断。学完之后，你将写出能随数据变化自动适应、经得起实际工作考验的公式。

## 准备工作

开始前，请确认以下条件：

- **软件：** Microsoft 365 桌面版或 Excel 2024/2021。Excel Online 也可用，但部分高级功能需要桌面版。
- **操作系统：** Windows 10/11 或 macOS 12+。
- **示例数据：** 从 [微软官方示例数据页面](https://learn.microsoft.com/en-us/power-bi/create-reports/sample-dataset) 下载练习工作簿，或自行创建包含以下列的数据：`日期`、`产品`、`类别`、`数量`、`单价`、`总价`、`区域`。
- **前置知识：** 熟悉 Excel 基本操作（选择单元格、复制粘贴）。无需公式经验。
- **设置：** 确保 **公式记忆式键入** 功能已开启（文件 > 选项 > 公式 > 公式记忆式键入，默认开启）。将 **计算选项** 设置为自动（公式选项卡 > 计算选项 > 自动）。

## 核心工作流

### 第一步：理解单元格引用（相对引用 vs 绝对引用）

**为什么重要：** 你写的每个公式在复制时都会调整引用。搞错这一点是公式错误的第一大原因。

- **相对引用（`A1`）：** 复制时自动调整。如果 B1 中的公式是 `=A1+1`，复制到 B2 就变成 `=A2+1`。
- **绝对引用（`$A$1`）：** 始终固定不变。把 `=$A$1+1` 复制到任何位置，它都指向 A1。
- **混合引用（`$A1` 或 `A$1`）：** 只锁定列或只锁定行。

**切换方法：** 选中公式编辑栏中的单元格引用，按 **F4** 循环切换四种模式。

**预期结果：** 公式向下或向右填充后，计算结果依然正确。

**停止条件：** 你能在一个新单元格中看一个公式，就能准确预测它引用了哪些单元格。

### 第二步：构建查找公式——VLOOKUP 和 XLOOKUP

**为什么重要：** 查找函数是 Excel 中使用频率最高的函数类别。它们根据关键词在表中找到对应的值。

**VLOOKUP 语法：**
```excel
=VLOOKUP(查找值, 表格区域, 返回列号, [匹配方式])
```

**示例——根据产品 ID 查找价格：**
```excel
=VLOOKUP("P1001", A2:C100, 3, FALSE)
```
在 A 列（A2:C100 的第一列）中搜索 "P1001"，返回该范围第 3 列的值。`FALSE` 强制精确匹配。

**XLOOKUP 语法（Excel 2021+/365）：**
```excel
=XLOOKUP(查找值, 查找数组, 返回数组, [未找到时返回], [匹配模式], [搜索模式])
```

**示例——查找产品价格并优雅处理错误：**
```excel
=XLOOKUP("P1001", A2:A100, C2:C100, "未找到", 0, 1)
```
如果产品不存在，返回 "未找到" 而不是 `#N/A`。

**为什么 XLOOKUP 更好：** 它可以向左或向右查找（VLOOKUP 只能向右查找），优雅处理缺失值，不会因为插入列而崩坏。

**停止条件：** 你能为任何有效的关键词构建一个能正确返回值的查找公式，并优雅处理不存在的键。

### 第三步：条件聚合——SUMIF、COUNTIF 和 AVERAGEIF

**为什么重要：** 原始数据本身说明不了问题。条件聚合函数可以汇总满足特定条件的数据。

**SUMIF 语法：**
```excel
=SUMIF(条件区域, 条件, [求和区域])
```

**示例——计算 A 产品的总收入：**
```excel
=SUMIF(B2:B1000, "产品A", G2:G1000)
```

**COUNTIF 语法：**
```excel
=COUNTIF(条件区域, 条件)
```

**示例——统计超过 500 元的销售笔数：**
```excel
=COUNTIF(G2:G1000, ">500")
```

**AVERAGEIF 语法：**
```excel
=AVERAGEIF(条件区域, 条件, [求平均值区域])
```

**示例——计算华东区域的平均订单金额：**
```excel
=AVERAGEIF(H2:H1000, "华东", G2:G1000)
```

**在条件中使用单元格引用：**
```excel
=SUMIF(B2:B1000, D1, G2:G1000)
```
当 D1 中包含 "产品A" 时，这与硬编码版本的结果相同，但现在你可以更改 D1 来查询任意产品。

**停止条件：** 你可以在不手动筛选的情况下，根据任意单个条件计算合计、计数和平均值。

### 第四步：文本函数——清洗和提取字符串

**为什么重要：** 真实世界的数据总是很凌乱。名字里有多余空格，产品代码需要拆分，地址需要重新格式化。

**TRIM——删除多余空格：**
```excel
=TRIM(A2)
```
删除前导、尾随和单词之间的多余空格。

**LEFT / RIGHT / MID——提取子字符串：**
```excel
=LEFT(A2, 4)    ' 前 4 个字符
=RIGHT(A2, 2)   ' 后 2 个字符
=MID(A2, 3, 5)  ' 从第 3 位开始取 5 个字符
```

**CONCATENATE / TEXTJOIN——合并文本：**
```excel
=TEXTJOIN(" ", TRUE, A2, B2, C2)
```
用空格连接文本，跳过空单元格。

**示例——从产品代码 "PRD-2024-001" 中提取年份：**
```excel
=MID(A2, 5, 4)
```

**停止条件：** 你能在 2 分钟内用公式清洗一个杂乱的数集（多余空格、不一致的大小写、混合格式）。

### 第五步：日期函数——计算时长和截止日期

**为什么重要：** 日期运算比看起来复杂得多。月份长度不同，简单的减法无法计算工作日历。

**TODAY 和 NOW——当前日期/时间：**
```excel
=TODAY()    ' 返回今天的日期，每天自动更新
=NOW()      ' 返回当前日期和时间
```

**DATEDIF——日期差：**
```excel
=DATEDIF(A2, B2, "d")    ' 间隔天数
=DATEDIF(A2, B2, "m")    ' 完整月份数
=DATEDIF(A2, B2, "y")    ' 完整年数
```

**WORKDAY——计算工作日：**
```excel
=WORKDAY(A2, 10, $F$2:$F$10)   ' A2 之后的 10 个工作日，排除 F2:F10 中的假日
```

**示例——根据出生日期计算年龄（年）：**
```excel
=DATEDIF(A2, TODAY(), "y")
```

**示例——距离截止日期的天数（负值表示已逾期）：**
```excel
=B2 - TODAY()
```
将结果单元格格式设置为 **常规**（而非日期）以显示数字。

**停止条件：** 你能计算两个日期之间的工作日数量，给日期加上 N 个工作日，并识别出逾期项目。

### 第六步：IF 嵌套和 IFS——决策逻辑

**为什么重要：** 业务规则很少是二元的。你需要根据多个条件分配等级、类别或状态。

**嵌套 IF 语法（传统方式）：**
```excel
=IF(条件1, 结果1, IF(条件2, 结果2, IF(条件3, 结果3, 默认值)))
```

**示例——根据订单总额分配折扣等级：**
```excel
=IF(G2>=1000, "金卡", IF(G2>=500, "银卡", IF(G2>=100, "铜卡", "标准")))
```

**IFS 函数（Excel 2019+/365）——更简洁的替代方案：**
```excel
=IFS(G2>=1000, "金卡", G2>=500, "银卡", G2>=100, "铜卡", TRUE, "标准")
```
最后的 `TRUE` 充当 "否则" 子句。

**示例——AND 条件判断：**
```excel
=IF(AND(G2>=500, H2="华东"), "优先", "普通")
```

**停止条件：** 你能在一个公式中编码多层次的业务规则（如绩效评级、折扣等级、优先级）。

## 故障排查

### 1. 公式以文本形式显示，不计算结果
- **现象：** 单元格显示 `=A1+B1` 的公式文本。
- **原因：** 单元格格式被设为文本，或公式前有多余的单引号。
- **解决：** 选中单元格，将格式改为常规（开始 > 数字 > 常规），按 F2 再按 Enter。如有单引号则删除。

### 2. VLOOKUP 返回 #N/A
- **现象：** 精确匹配的 VLOOKUP 返回 `#N/A`，但看起来值确实存在。
- **原因：** 数据有前导/尾随空格、数字以文本格式存储、或查找列不是区域的第一列。
- **解决：** 使用 `=TRIM(A2)` 处理查找值。使用 `=VALUE(A2)` 转换文本型数字。确认 table_array 的首列就是查找列。

### 3. 日期计算出现 #VALUE! 错误
- **现象：** 日期相减出现 `#VALUE!` 而不是数字。
- **原因：** 其中一个或两个 "日期" 实际上是文本字符串，而非 Excel 日期序列号。
- **解决：** 使用 `=DATEVALUE(A2)` 转换文本日期。使用 `=ISNUMBER(A2)` 确认单元格是否包含数字（Excel 中的日期本质上是数字）。

### 4. SUMIF 返回 0
- **现象：** 条件求和返回 0，但明明有匹配的值。
- **原因：** 条件区域和求和区域未对齐（行列数不一致），或条件中比较运算符使用不当。
- **解决：** 确保两个区域行数相同。对于带运算符的文本条件，使用：`=SUMIF(A:A, ">="&"产品", B:B)`。

### 5. 循环引用警告
- **现象：** Excel 显示"存在一个或多个循环引用"。
- **原因：** 公式直接或间接引用了自身所在的单元格。
- **解决：** 转到公式 > 错误检查 > 循环引用，定位问题单元格。移除自引用。如有意使用迭代计算，在文件 > 选项 > 公式中启用"启用迭代计算"。

## 专家技巧

**时间节省：** 某财富 500 强公司的销售分析师通过用 SUMIFS 公式替代手动筛选，将每周报告时间从 4 小时缩减到 25 分钟。学习这些函数的一次投入在第一周就回本了。

**命名范围让公式更清晰：** 与其写 `=SUMIF(B2:B1000, "产品A", G2:G1000)`，不如定义命名范围：选中 B2:B1000，转到公式 > 定义名称，输入 "产品"。然后写：`=SUMIF(产品, "产品A", 总计)`。命名范围让公式自文档化，且不会因插入行而崩坏。

**Excel 表格（Ctrl+T）：** 将数据区域转换为表格（Ctrl+T）。引用表格列的公式——如 `=SUMIF(表1[产品], "A", 表1[总计])`——会在添加行时自动扩展。这是构建弹性电子表格的最重要习惯。

**F4 快速填充：** 输入 `=VLOOKUP(...)` 一次后，选中单元格，双击填充柄（右下角）即可将公式复制到相邻数据的最后一行。Excel 会自动停止。

**公式求值：** 使用公式 > 公式求值，一步步查看复杂公式的每个计算环节。按 **Ctrl+`**（反引号）可一次性显示工作表中所有公式。

**通配符与 IF：** `=COUNTIF(A:A, "*华北*")` 统计 A 列中所有包含 "华北" 的单元格。通配符同样适用于 SUMIF、COUNTIF、AVERAGEIF，但不能直接用于 SUMIFS/COUNTIFS（需将通配符放入条件中）。

## 总结

在本教程中，你学习了每位 Excel 用户都需要掌握的六种核心公式模式：
- **单元格引用**（相对、绝对、混合）——公式弹性的基础
- **查找函数**（VLOOKUP 和 XLOOKUP）——按关键词查找数据
- **条件聚合**（SUMIF、COUNTIF、AVERAGEIF）——带条件的数据汇总
- **文本函数**（TRIM、LEFT、TEXTJOIN）——清洗杂乱数据
- **日期函数**（DATEDIF、WORKDAY、TODAY）——计算时间间隔
- **决策逻辑**（IF、IFS、AND）——编码业务规则
